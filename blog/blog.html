closures

February 17, 2018

One of the biggest questions that got me stuck in my most recent phone interview was closure. Not the relationship type of closure, but JavaScript closure. I told the interviewer in an act of nervousness that closurse were "inner functions that have access to global variables." That wasn't very correct.


var addTo = function(passed) {
    var add = function(inner) {
        return passed + inner;
    };
    return add;
};

var addThree = new addTo(3);
var addFour = new addTo(4);

console.dir(addThree); // Closure is 3
console.dir(addFour); // Closure is 4

// Closure preserves a value inside of the function; closures are functions with preserved data

console.log(addThree(1));
console.log(addFour(1));
